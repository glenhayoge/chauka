# Navigating the Interface

Chauka's interface is organised around your logframe. Once you select a logframe, you'll see a set of tabs that give you access to different aspects of your programme.

## The main navigation flow

After signing in, you follow this path:

1. **Select your organisation** — if you belong to more than one
2. **Select a logframe** — you'll see all logframes you have access to
3. **Work within the logframe** — using the tab bar along the top

## Logframe tabs

Once inside a logframe, you'll see these tabs:

| Tab | Purpose |
|---|---|
| **Dashboard** | At-a-glance summary of logframe progress, key metrics, and recent activity |
| **Overview** | High-level view of the entire results framework and its status |
| **Design** | Build and edit the results tree — add Impacts, Outcomes, Outputs, indicators, and assumptions |
| **Monitor** | Enter data against indicators, view progress towards targets |
| **Budget** | Track budget lines and expenses against activities |
| **Workload** | View and manage activities, milestones, and timelines |
| **People** | See team members working on this logframe and their roles |
| **Settings** | Configure logframe options — reporting periods, result levels, permissions |

## Key interface patterns

### Expanding and collapsing results

The results tree in the Design and Monitor tabs is collapsible. Click on a result to expand it and see its children, indicators, and activities. This helps you focus on the part of the logframe you're working on.

### Inline editing

Many fields in Chauka can be edited directly. Click on text fields, descriptions, or values to edit them in place without navigating to a separate form.

### Status indicators

Chauka uses coloured status dots throughout the interface:

- Results, indicators, and activities show progress status
- Ratings help you quickly assess whether things are on track
- Risk ratings on assumptions show which risks need attention

### Navigation breadcrumbs

When you're deep in a logframe, breadcrumbs at the top help you understand where you are and navigate back up the hierarchy.

## Your profile

Click your name or avatar to access your profile, where you can:

- Update your display name and email
- Change your password
- View the organisations you belong to
