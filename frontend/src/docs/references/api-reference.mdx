# API Endpoints

Complete list of Chauka API endpoints. All endpoints except `/api/auth/token` and `/health` require JWT authentication.

> Interactive API documentation is available at `/docs` (Swagger UI) and `/redoc` when the backend is running.

## Authentication

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/auth/token` | Login (form-urlencoded) — returns JWT |

## Users

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/users` | List all users |

## Organisations

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/organisations` | List user's organisations |
| POST | `/api/organisations` | Create organisation |
| GET | `/api/organisations/{id}` | Get organisation |
| PATCH | `/api/organisations/{id}` | Update organisation |

## Programs

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/organisations/{org_id}/programs` | List programs |
| POST | `/api/organisations/{org_id}/programs` | Create program |

## Logframes

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/logframes` | List user's logframes |
| POST | `/api/logframes` | Create logframe |
| GET | `/api/logframes/{id}` | Get logframe |
| PATCH | `/api/logframes/{id}` | Update logframe |
| GET | `/api/logframes/{id}/bootstrap` | Get full logframe data |
